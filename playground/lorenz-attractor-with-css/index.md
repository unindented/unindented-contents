Drawing a [Lorenz attractor](https://en.wikipedia.org/wiki/Lorenz_system) with CSS shadows (generated by [Sass](https://sass-lang.com/)), no JavaScript involved.

You really don't want to see the resulting CSS though. It's a huge `text-shadow` property with the coordinates for each of the 5000 iterations of the equations that Sass ran.
